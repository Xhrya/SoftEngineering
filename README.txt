To run the GPT communicator server, please run gptserver.js. 
To communicate with the server on a back-and-forth, please run communicatortest.js. 
To send POST requests, use communicator.js. 
You will need to provide your own GPT API key in a file called cred.env. 
To run jasmine tests, please run: 
jasmine src/chatgpt/gptserver.spec.js
The encrypt.js file is NOT needed under normal operation, it is only needed to update the GPT program specifications.
